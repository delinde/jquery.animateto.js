jQuery.animateTo.js

Usage

It is pretty simple, prepare your markup and initialize "animateTo".

$('.product').animateTo('#cart');

Options

  • clone
    Clone the element or use it itself. (default: true)
  • useTransition
    Use css3 transitions or jQuery's builtin animate-method. (default: false)
  • duration
    The animation speed. (default: 900)
  • easing
    The easing method you want to use. (default: 'swing')
  • extendAnimation
    Object to extend the cloned elements css-styling. (default: {opacity: 0})
